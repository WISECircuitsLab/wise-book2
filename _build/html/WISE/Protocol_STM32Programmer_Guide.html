BioMADE Electrochemical Bioreactor Monitoring Guide

Protocol STM32Programmer_Guide#

Software:

  1. Install Arduino IDE if you haven’t already
  2. Install STM32CubeProgrammer
    1. This should show up as a folder called “STMicroelectronics.” → Move this to your application folder for easy access
    2. The path inside this folder should be as follows: STMicroelectronics → STMCube → STM32CubeProgrammer → here, the actual application will reside
  3. Install STM32CubeIDE (most likely won’t need, but download anyway)

Hardware:

  1. MCU of use
    1. Always try to program through Arduino first (ensure that Arduino supports your MCU; if not, use the following method)
  2. ST LINK (not needed if using Arduino IDE)
  3. USB/USB-C Cable

Set Up:

  1. Using jumper wires, make the following connections from the ST Link to the MCU
    1. SWDIO → PA13 (for the STM32WLE5JC), check the pinout diagram if a different MCU
    2. SWDCLK → CLK
    3. VDD → 3V3 or 5V → STM32WLE5JC uses 3V3, check data sheet if something different
    4. GND → GND
    5. Reset → Reset
  2. Once the connections to the MCU are established, open the STM32CubeProgrammer Application. You should see the following:
  3. On the right side, under the ST-Link Configuration, click the refresh button to the right of the serial number drop-down
    1. Your ST Link serial number should appear, and fill in the other information. The image below shows what it should look like:
  4. Now to connect
    1. Either hit the reset button or ground the reset pin
    2. Hold the reset, then hit the green “Connect” button
    3. Then release the reset
    4. YOU SHOULD BE CONNECTED NOW
  5. Once connected, go to the left sidebar to “OB”
  6. Here, expand the Read Protection tab and change the read protection to AA. This removes it.
  7. Then hit “Apply”
  8. Navigate to this symbol for “Erasing and Programming”
  9. Now open Arduino IDE with your desired program
  10. Go to Sketch → “Export Compiled Binary”
  11. Go back to the STM32CubeProgrammer
  12. Under the drop-down down navigate to the path with your Arduino IDE files and the specific sketch, and select the corresponding hex file
  13. Select the same programming configurations as in the image above, then select Start Programming
